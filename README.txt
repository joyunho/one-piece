원랜디 2.305 악몽 실전 판단 코치 v16.0.0
==========================================

v16은 현재 TMO 패에서 다음 한 행동을 고르고, 패가 바뀔 때마다 다시
계산하는 단일 화면 판단 도구입니다. 고정된 최종 9기나 클리어 확률을
약속하지 않습니다. v16부터는 증명 가능한 제작이 없어도 침묵하지 않고,
남은 필수 결손을 닫는 최근접 목표(부족 재료·필요 선위)를 제시합니다.

설치
----

1. ZIP을 원하는 폴더에 압축 해제합니다.
2. Chrome 또는 Brave에서 chrome://extensions 를 엽니다.
3. 개발자 모드를 켜고 `압축해제된 확장 프로그램을 로드`를 누릅니다.
4. `ord_tmo_auto_extension_v15_0_0_rebuild` 폴더를 선택합니다.
5. https://tmo.gg/ko/build-helper/32172 를 열고 새로고침합니다.
   기존 34366 도우미도 호환됩니다.
6. 확장 아이콘에서 실전 코치를 엽니다.

자동 연동을 쓰지 못하면 아래 파일을 직접 열어 수동 입력할 수 있습니다.
`ord_2305_nightmare_helper_v16_0_0_manual.html`

한 화면의 8개 영역
------------------

1. 다음 행동: 지금 만들 유닛, 리롤 한 장 또는 보류 판단
2. 다음 행동 미리 준비: 확정 행동 뒤 검토할 최대 2기
3. 현재 스펙: 물딜/마딜 경로와 실제 보유 역할 결손
4. 보유 희귀로 만들 수 있는 전설급: 개별 제작 가능한 후보 최대 6기
5. TMO 152킬 선택: 특별함 포함 희귀 완성도 예상
6. 오로성 선택: 이번 판 목표 이감·방깎·스턴
7. 상위 정보: 메인 상위와 현재 패 후보 최대 6개
8. 현재 라운드 + 녹화: 자동 라운드, 동기화, 결과, JSON 저장

152킬 예상값은 `원 TMO → 152 포함 예상`으로 분리합니다. 원본 TMO 수치를
덮어쓰지 않으며, 실제 특별함 보유가 감지되면 가상 증가분을 다시 더하지 않습니다.

스토리 등급
-----------

스토리 측정표는 희귀 / 상위 / 전설급 리그를 서로 섞지 않고 각 리그 안에서
`S, A, B, C, D, E, F` 7단계 등분위로 표시합니다. 원본 파괴 시간·퍼센트와
원본 순위는 그대로 보존합니다.

판단 원칙
---------

- 필수 역할표는 보유 희귀·특별의 직접 전투 역할까지 세는 live 기준이며,
  그런 희귀를 재료로 소비하는 제작은 결손 재개방이 그대로 보여 감점됩니다.
- 증명 가능한 제작이 없으면 결손별 최근접 후보(부족 재료·필요 선위)를
  회복 목표로 제시합니다. 위습 예산은 모든 경로에서 필수 결손의 최저
  선위 마감 후보를 예약합니다.
- 첫 희귀와 첫 전설·히든은 TMO 완성도 순서를 우선합니다.
- 152킬 특별함을 고른 첫 희귀는 레시피상 절약되는 선택 위습을 반영한 예상
  완성도로만 순위를 비교합니다.
- 상위 방향은 현재 패 기준 후보를 최대 6개만 보여주며, 미래 조합이 완성되지
  않아도 필요한 타이밍에는 방향을 확정할 수 있습니다.
- 레일리·해적선·특수 아이템 선행 유닛은 실제 보유 전 추천하지 않습니다.
  압살롬 예외는 유지합니다.
- 왜곡은 항상 허용하지만 높은 선위 비용, 같은 계보 중복, 핵심 재료 충돌을
  감점해 더 싼 동등 대안보다 먼저 고르지 않습니다.
- 물딜은 최소 0.5스턴과 상시 풀방깎을 먼저, 그다음 안전 이감·광보잡,
  충분한 스턴을 봅니다. 이감 초과 가점은 일반 102, 나스쥬로 117에서 멈춥니다.
- 마딜은 `2상위+토키`와 `1상위+단·끝`을 별도 경로로 계산합니다.
- 제작 또는 리롤 한 번마다 TMO 수량 변화를 확인한 뒤 다음 행동을 새로 계산합니다.

녹화와 한계
-----------

`녹화`는 영상이 아니라 관측 패, 추천 근거, 152 예상치, 사용자 선택, 제작 확인,
결과를 담는 로컬 판단 기록입니다. `녹화 JSON`을 저장해 대화에 첨부하면 판단
순서를 재생할 수 있습니다. 화면·계정·채팅 명령은 저장하거나 전송하지 않습니다.

이 도구는 실제 보스 DPS, 스킬 발동, 컨트롤, 도움소, 업그레이드와 랜덤 드랍을
측정하지 않습니다. 화면의 역할 충족은 50라 보스 처치나 65라 클리어 보장이
아닙니다.
