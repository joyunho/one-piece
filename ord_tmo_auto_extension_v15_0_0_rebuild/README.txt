원랜디 2.305 악몽 실전 판단 코치 v17.8.0 — 확장 프로그램
=============================================================

이 폴더를 chrome://extensions 의 `압축해제된 확장 프로그램을 로드`에서
선택하세요. 주 도우미는 TMO 32172이며 34366도 호환됩니다.

기본 화면은 다음 8개 영역만 표시합니다.

1. 다음 행동
2. 다음 행동 미리 준비
3. 현재 스펙
4. 보유 희귀로 만들 수 있는 전설급
5. TMO 152킬 선택·희귀 완성도 예상
6. 오로성 선택
7. 상위 정보
8. 현재 라운드 + 판단 녹화

- 첫 실제 TMO 유닛 감지 시 라운드 자동 시작
- 물딜 / 마딜과 마딜 세부 경로를 화면에서 선택
- 상위 후보와 제작 가능 전설급 후보는 각각 최대 6개
- 152 예상값은 원 TMO 수치와 분리하고 실제 보유 시 중복 계산 방지
- 왜곡 항상 허용, 고선위·계보 중복·재료 충돌 감점
- 일반 이감 102, 나스쥬로 117 이후 초과 가점 없음
- 특수 선행재료는 실제 보유 전 추천 차단(압살롬 예외)
- 스토리 등급은 리그별 SSS / SS / S / A / B / C / D / E / F 9단계(v17.7 개편)
- 판단·선택·제작 확인·결과를 로컬 `.ordlog.json`으로 기록

외부 서버나 Node.js는 필요하지 않습니다. 판단 녹화는 영상이 아닙니다.
실제 보스 DPS와 컨트롤은 측정하지 않으므로 역할 수치 충족을 클리어 보장으로
표시하지 않습니다.
